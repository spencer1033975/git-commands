clone- copies existing Git repo from a server into your device
checkout- switches between different branches in repo
-b -creates new branch and switches to it
status- shows files being changes
push- sends committed changes into repo
-u -links local branch to remote 
pull -merges changes from remote server to current branch
add -adds file changes to staging area
commit -saves staged changes to repo history
-m -allows you to commit message directly
-a -stages files that are modded/deleted
-am stages modded files and includes message
branch -lists, creates, deletes branches
-a -shows all branches
-d -deletes branch
merge- combines changes in branches
